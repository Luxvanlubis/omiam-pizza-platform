### 1️⃣ Créer le projet
```
https://supabase.com → New Project
┌─────────────────────┐
│ Project: omiam-pizza│
│ Region: us-east-1   │
│ Password: [secure]  │
└─────────────────────┘
```

### 2️⃣ Récupérer les clés
```
Settings → API
┌─────────────────────────────┐
│ URL: https://xyz.supabase.co│
│ Anon Key: eyJ...            │
│ Service Key: eyJ...         │
└─────────────────────────────┘
```

### 3️⃣ Configurer
```bash
# Dans votre terminal
node setup-env.js
# Suivez les instructions
```

### 4️⃣ SQL
```sql
-- Copier/coller dans SQL Editor
copy supabase-schema.sql → Run
```

### 5️⃣ Tester
```bash
npm run dev
# http://localhost:3000 ✅
```
